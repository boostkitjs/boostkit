Add `@rudderjs/cashier-paddle` — Paddle billing for RudderJS apps. Mix `Billable` into your User model for checkout sessions, subscription state, refunds, pricing previews, and a signed webhook receiver. Includes Prisma schema fragment, React checkout button + inline checkout components, and `cashier:install` / `cashier:webhook` / `cashier:sync` CLI commands.
